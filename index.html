Galaxy Generator

Project Description: 3D galaxy generator using Three.js, capable of creating over 3,00,000 unique galaxy shapes. The project utilizes numerous trigonometric functions for intricate star positioning, enhancing the visual appeal of the galaxies. A dynamic user interface allows for real-time parameter manipulation, providing users with the ability to customize their galaxies. Additionally, a screenshot feature was implemented, enabling users to save images of their custom galaxies.

3D Portal Scene

Project Description: A 3D portal scene was created using Blender, demonstrating 3D modeling and scene creation skills. This was followed by baking the textures for the 3D model in Blender, which showcased advanced texturing techniques. Custom shaders were then implemented for the portal light and fireflies effects, demonstrating GLSL and shader programming skills. Finally, the 3D scene was integrated into a web environment using Three.js, and user interaction was provided with OrbitControls.
